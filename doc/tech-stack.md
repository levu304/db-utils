# Technology Stack

## 1. TypeScript
- Primary programming language for development
- Provides type safety, better tooling, and maintainability
- Enables cross-platform compatibility with JavaScript

## 2. Pnpm
- Package manager for efficient dependency management
- Faster installs and smaller disk usage compared to npm
- Supports workspace monorepos for multi-package projects

## 3. PostgreSQL
- Relational database management system
- Used for both source and destination databases
- Supports complex queries and data integrity features

## 4. Node.js (implicit)
- Runtime environment for CLI tool execution
- Provides asynchronous I/O capabilities for database operations

## 5. Moon (Monorepo Tool)
- Used for managing multiple packages within this project
- Provides shared dependencies, improved build performance, and better workspace management

## 6. Development Tools
- VS Code (recommended editor with TypeScript support)
- Git for version control (via .gitignore configuration)
- Optional: Docker for containerized database testing
